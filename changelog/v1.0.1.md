# v1.0.1版本更新内容
    - 修复了一些bug
    - 改进了任务排序无截止日期的（依据重要度）排序
    - 添加了每项任务的剩余时间显示
    - 添加了设置选项，包括：
        - 设置自动更新间隔（默认3600秒）
    - 新增任务类别
    - 新增任务标签
    - 新增搜索筛选功能
        - 按任务名称搜索
        - 按任务类别搜索
        - 按任务标签搜索
        - 按任务紧急度搜索
        - 按任务重要度搜索
        - 按任务状态搜索（未完成/已完成）
        - 按任务截止日期搜索（今天/未来7天）
        - 按任务创建时间搜索（今天/未来7天）
    - 优化了用户界面布局，避免信息遮挡
    - 优化页面布局（防止有数据时遮挡）
    - 新增任务统计功能模块
        - 每日/每周/每月完成任务数量趋势
        - 不同类别的任务分布
        - 不同标签的任务分布
        - 任务完成率和平均时间统计
        - 图表展示
            - 饼图
            - 条形图
